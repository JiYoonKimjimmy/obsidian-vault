#### 개선 사항
- 3차 개선으로 진행했던 기존 가맹점 1000건 IN 절 조회 로직 제거
	- Thread-Pool Size 2개 기준 `4차 개선` vs `5차 개선` 비교하여 결정
	- 기존 가맹점 정보 단건 조회 변경 후 업데이트 방식으로 로직 변경
- Thread-Pool Size 프로퍼티 설정하도록 변경
	- 운영 서버 테스트 후 Size 설정 예정
