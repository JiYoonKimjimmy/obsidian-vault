
## 문의 사항

- `CARD_MASTER(카드원장)` 테이블 `STATUS_CD != '2(정상)'` 조건의 카드 중 카드잔액 정보가 존재 가능 여부 확인 요청
- `POLICY_GOODS_GROUP(정책상품그룹)`  정의서 형식 확인 요청
	- 일부 필드명 작성 누락 또는 잘못된 샘플 데이터 작성인지 확인 요청
- `MERCHANT_DTL_INFO(직승인가맹점)` 테이블 컬럼 중 가맹점주 회원 `CI연게정보` 포함 가능 확인 요청
	- 가맹점주 `CI연계정보` 가 없다면, 가맹점 거래 내역을 가맹점주들이 조회할 수 있는 방법 확인 요청
- `MERCHANT_DTL_INFO(직승인가맹점)` 테이블 `COMPANY_REGNUM`. `ACCOUNT_NUMBER`, `BANK_MEMBER_NAME` 최종 데이터 전달은 마스킹 아닌 원문인지 확인 요청
- `CARD_RECOV(거래원장)` 테이블 `TRANS_TYPE(거래종류)` 중 `교환` 거래에 대한 설명 요청
