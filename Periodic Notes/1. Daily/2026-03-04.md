## mupay 계좌 등록 결과 Event 처리 프로세스 개발

1. `mupay` 계좌 등록 결과 Event 수신
2. `event.partnerBankCode == "BK89"` 인 경우, 케이뱅크 제휴 계좌 등록 결과 처리 진행
	1. `hashCi` 기준 `KbankCustomer` 정보 조회
	2. `event.result` 결과에 따른 케이뱅크 제휴 계좌 등록 처리 진행
		1. `event.result == "SUCCESS"` 인 경우, 케이뱅크 제휴 계좌 등록 성공 처리 진행
			1. `KbankCustomer.processStatus = "COMPLETED"` 변경 처리
			2. `KbankCustomer.status = "LINKED"` 변경 처리
		2. `event.result == "FAILED"` 인 경우, 케이뱅크 제휴 계좌 등록 실패 처리 진행
			1. `KbankCustomer.processStatus = "OPEN_BANK_AGREE_PROCESSING"` 변경 처리
	3. 정보 변경 완료한 `KbankCustomer` DB 저장 Event 발행 처리
