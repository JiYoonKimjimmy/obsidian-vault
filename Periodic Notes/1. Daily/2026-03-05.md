## PREPAY-1048

- `MoneyTokenApiControllerV1.getSessionByToken` 함수 기준 신규 기능 추가하여 V2 API 신규 개발
- V2 API 추가 개발 사항
	- 케이뱅크 연동 여부 확인: `hashCi` 정보 기준으로 `KbankCustomer` 조회하여 `processStatus != 'AFFILIATED_PROCESSING' & status == 'LINKED'` 인 경우, `true` 응답 처리
	- 오픈뱅킹 등록 여부 확인: `hashCi` 정보 기준으로 `KbankCustomer` 조회하여 `processStatus == 'COMPLETED'` 인 경우, `true` 응답 처리


| isKbankLinked | AFFILIATED_PROCESSING | AFFILIATED_COMPLETED | OPEN_BANK_AGREE_PROCESSING | COMPLETED |
| ------------- | --------------------- | -------------------- | -------------------------- | --------- |
| PROCESSING    | false                 | true                 | true                       | true      |
| LINKED        | false                 | true                 | true                       | true      |
| UNLINKED      | false                 | false                | false                      | false     |

| isOpenBankingRegistered | AFFILIATED_PROCESSING | AFFILIATED_COMPLETED | OPEN_BANK_AGREE_PROCESSING | COMPLETED |
